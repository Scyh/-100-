Set：
            1. 成员不重复；
            2. 只有键值，没有键名；
            3. 支持遍历；
            4. 方法：add、delete、has
        
        WeakSet：
            1. 成员只接受对象；
            2. 成员都是弱引用（不计入垃圾回收机制）
            3. 不能遍历；
            4. 方法：add、delete、has

        Map：
            1. 值-值对应（可以使用对象等作为键名）；
            2. 支持遍历；
            3. 方法：set、get、clear、has、clear、keys、values、entriers、forEach；
            4. 属性：size

        WeakMap：
            1. 只接受对象作为键名；
            2. 键名所指的对象，都是弱引用（不计入垃圾回收机制）；
            3. 不支持遍历；
            4. 方法：get、set、has、delete
