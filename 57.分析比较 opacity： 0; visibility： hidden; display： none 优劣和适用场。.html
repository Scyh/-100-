display: none
            会引起 回流
            元素不占据文档流
            
        opacity: 0
            只会引起重绘
            仍然占据文档流
            如果有子元素，子元素也会消失不见
            可以点击到元素

        visibility: hidden
            只会引起重绘
            仍然占据文档流
            如果有子元素，并且子元素设置为visibility: visible，子元素可见
            点击不到元素
