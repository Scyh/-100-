模块化开发：

        AMD：
            依赖前置，必须提前声明
            异步加载

            define(['xxx.js'], function(modulexxx) {
                var a = function() {}
                return a
            })
        
        CMD：
            依赖用到时，再声明，
            依赖在模板执行时才会加载执行
            define(function(require, exports, module) {
                var moduleA = require('./moduleA'); // 同步

                require.async('./moduleB', function(moduleB) {

                });

                var moduleCmd = function() {}

                对外暴露
                exports.moduleCmd = moduleCmd
            },)
            

        commonjs：
            nodejs的模块化
                服务端：
                    运行时，同步加载

                客户端：
                    需要提前编译处理。

                var a = require('./a');

                var b = 'SOME VARIABLE'
                modules.export = b

        ES6：
            输出的是值引用
            编译时输出
            import
            export

            export default 实质是导出一个名为 defalut 的变量，并将导出的值赋值到default。并且系统允许为它取任何名字
